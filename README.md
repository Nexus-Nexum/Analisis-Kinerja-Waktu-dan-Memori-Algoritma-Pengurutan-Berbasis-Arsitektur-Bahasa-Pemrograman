# Benchmark Sorting Algorithm (C++, Python, Node.js)

Repositori ini berisi kode sumber untuk penelitian perbandingan kinerja algoritma sorting.

## Struktur File
* `generate_data.py`: Skrip untuk membuat dataset angka acak (.txt).
* `test_cpp.cpp`: Kode benchmark untuk C++.
* `test_python.py`: Kode benchmark untuk Python.
* `test_node.js`: Kode benchmark untuk Node.js.
* `data_*.txt`: Contoh dataset hasil generate.

## Cara Menjalankan
1. Jalankan `generate_data.py` terlebih dahulu untuk membuat dataset.
2. Jalankan file test masing-masing bahasa.
